## Step 1 — Connect Your First Server

AreaWP manages servers via a lightweight agent. Install the agent on any Linux server you want to manage.

<Steps>
  <Step title="Go to Servers → Agent Fleet">
    In the left sidebar, navigate to **Servers** → **Agent Fleet**.
  </Step>

  <Step title="Click Add Server">
    Click **Add Server** and enter a name for your server.
  </Step>

  <Step title="Copy the install command">
    AreaWP generates a one-line install command with your panel URL and a registration token.

    ```bash
    curl -fsSL https://your-panel.com/install.sh | sudo bash -s -- \
      --token "sk_reg_YOUR_TOKEN" \
      --server "https://your-panel.com"
    ```
  </Step>

  <Step title="Run it on your server">
    SSH into the target server and run the command. The agent installs as a systemd service and connects to your panel automatically.
  </Step>

  <Step title="Verify connection">
    The server appears as **Online** in the Agent Fleet list within seconds.
  </Step>
</Steps>

## Step 2 — Deploy Your First Site

Once a server is connected, you can deploy WordPress sites and web applications.

<Tabs>
  <Tab title="WordPress Site">
    1. Navigate to **WordPress** in the sidebar
    2. Click **New Site**
    3. Select the target server
    4. Enter the domain name (e.g. `mysite.com`)
    5. Choose a WordPress template or start fresh
    6. Click **Deploy**

    AreaWP will:
    - Create a Docker container for the site
    - Install WordPress and configure the database
    - Set up Nginx with SSL (if domain resolves)
    - Have the site live within 60–90 seconds
  </Tab>

  <Tab title="From a Template">
    1. Navigate to **Services** → **Templates**
    2. Browse available app templates
    3. Click **Deploy** on your chosen template
    4. Select target server and enter domain
    5. Fill in any template variables
    6. Click **Deploy**
  </Tab>
</Tabs>

## Step 3 — Set Up Billing (Optional)

If you're selling hosting to clients, configure billing next.

<Steps>
  <Step title="Create a Pricing Plan">
    Go to **Administration → Pricing Plans** and click **New Plan**. Set the name, price, billing period, and resource limits (sites, backups, etc.).
  </Step>

  <Step title="Enable a Payment Gateway">
    Go to **Administration → Payment Gateways**. Enable **Stripe** (for credit cards) or **Bank Transfer** and enter your credentials.
  </Step>

  <Step title="Invite your first client">
    Go to **Administration → Users** and click **Invite User**. The client receives an email with a registration link.
  </Step>
</Steps>

---

## Step 4 — Configure Branding

Make the platform your own with white-label settings.

1. Go to **Administration → Settings** → **White Label**
2. Set your brand name, upload your logo
3. Choose **Text only** mode (replaces "AreaWP" with your brand name) or **Full logo** mode
4. Save — changes take effect immediately for all users
