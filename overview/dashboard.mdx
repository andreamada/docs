</Info>

## Metric Cards

The top row of the dashboard shows key counters:

| Metric | Description |
|--------|-------------|
| **Servers** | Total connected agents (online / offline) |
| **Sites** | Total deployed applications |
| **Users** | Registered client accounts |
| **Backups** | Recent backup operations |
| **Subscriptions** | Active client subscriptions |
| **Alerts** | Open security or health alerts |

---

## Server Health

The server health section lists all connected agents with their real-time resource usage:

- **CPU usage** — percentage across all cores
- **RAM usage** — used / total memory
- **Disk usage** — used / total storage
- **Load average** — 1m, 5m, 15m averages
- **Uptime** — time since last reboot
- **Status** — Online / Offline / Warning

Click any server row to navigate to that server's detail page.

---

## Recent Activity

The activity feed shows the last 20 actions performed on the platform:

- Site deployments and deletions
- Backup completions
- User registrations and login events
- Security alerts (failed logins, blocked IPs)
- Subscription changes and payments
- System tasks and cron job executions

<Tip>
For full activity history, navigate to **Administration → Activity Overview** (`/activity`).
</Tip>

## Quick Actions

The dashboard toolbar provides shortcuts for the most common tasks:

| Action | Description |
|--------|-------------|
| **New Site** | Deploy a new WordPress site or app |
| **Add Server** | Register a new managed server |
| **Invite User** | Send a client invitation email |
| **Create Backup** | Trigger a manual backup |

---

## Platform Mode

The dashboard header shows whether you are in **SaaS** or **WaaS** mode. This affects which features are visible and how billing is configured. Change the platform mode in **Administration → Settings → Platform**.

### Sidebar Presets

Admins can switch between sidebar presets to show only the features relevant to their workflow:

| Preset | Shows |
|--------|-------|
| **Full** | Everything |
| **Web Hosting** | Domains, SSL, databases, core web tools |
| **Email Admin** | Email server, DNS, monitoring, security |
| **Docker / DevOps** | Docker, Git, monitoring, workflow |
| **Minimal** | Dashboard, servers, terminal only |

Customize presets via **Settings → Sidebar** or from the user menu at the bottom of the sidebar.
