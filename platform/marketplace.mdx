## Browsing the Catalog

The top of the page shows type toggle buttons: **All**, **Plugins**, **Themes**.

Two tabs:

### Browse Tab

Grid of extension cards. Each card shows:

| Element | Description |
|---------|-------------|
| **Screenshot** | Plugin/theme preview image |
| **Type badge** | Plugin or Theme |
| **Category badge** | e-commerce, SEO, security, etc. |
| **Name & Author** | Display name and developer |
| **Description** | Short summary |
| **Version** | Current version number |
| **Price** | Free or paid amount |
| **Action button** | Install / Buy / Renew |

**Search bar** and **category filter** narrow results.

### Installed Tab

Two sections:

**Pending** — extensions awaiting approval (submitted but not yet approved by platform admin), with approval status indicator.

**Active** — installed extensions showing:
- Price and renewal type (one-time / monthly / yearly)
- Expiry label
- Action buttons: Reinstall, Uninstall, Renew, Cancel Subscription, Revoke Cancellation

---

## Installing an Extension

**Free extensions:** Click **Install** → select the target WordPress site in the **Site Select Modal** → confirm.

**Paid extensions:** Click **Buy** → **Payment Modal** opens:
1. Select payment method (Stripe, PayPal, Bank Transfer, etc.)
2. Select the target WordPress site
3. Complete payment — for card payments, processed immediately; for Bank Transfer, marked pending

After successful purchase, the extension appears in the **Installed → Active** section.

---

## Subscription-Based Extensions

Some extensions use monthly or yearly subscription pricing:

- **Renew** — extend the subscription before it expires
- **Cancel Subscription** — stop auto-renewal (access continues until expiry)
- **Revoke Cancellation** — undo a pending cancellation before the period ends

---

## Admin — Submitting Templates

Click **Submit Plugin/Theme** (top-right, admin only) to add a custom extension to the catalog:

- Name, description, category, version, and price
- Upload a ZIP file
- Set visibility: public (visible to all users) or private (admin-only)

Use **Edit** and **Delete** to manage your submitted extensions.

---

## Payment Return

When returning from an external payment gateway (PayPal, Stripe redirect), the Marketplace shows a **Payment Return** screen confirming the purchase status before returning to the catalog.
