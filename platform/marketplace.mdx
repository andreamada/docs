## WordPress Plugins

Browse and manage the plugins available for client sites. Plugins in the marketplace are:

- **Pre-licensed** — Covered by your platform license for all client sites
- **Auto-updated** — AreaWP keeps plugins updated across all sites
- **One-click install** — Install on any site from the UI

### Installing a Plugin on a Site

1. Browse to **Marketplace → WordPress Plugins**
2. Click **Install** on the desired plugin
3. Select one or more sites
4. Click **Confirm** — plugin is installed and activated

### Bulk Plugin Management

From the site list, select multiple sites and apply plugin operations:

- Install plugin on all selected sites
- Update to latest version
- Activate / Deactivate
- Remove

## WordPress Themes

Manage the theme library available for client sites:

| Column | Description |
|--------|-------------|
| **Theme name** | Display name |
| **Version** | Current version |
| **Sites using it** | How many client sites have this theme |
| **Status** | Available / Update available / Deprecated |

### Client Theme Access

Control which themes each pricing plan can use:

1. Go to **Administration → Pricing Plans**
2. Select a plan → **Features → Theme Library**
3. Choose: All themes, Basic themes only, or specific themes

## AreaWP Extensions

Extend the AreaWP panel with add-on modules:

| Extension | Description |
|-----------|-------------|
| **MainWP Bridge** | Sync with MainWP for advanced WordPress management |
| **Cloudflare Integration** | Automatic Cloudflare DNS and cache management |
| **Uptime Robot** | Alternative uptime monitoring via Uptime Robot |
| **WHMCS Bridge** | Sync clients and billing with WHMCS |
| **Agency Tools** | Bulk white-label reporting for agencies |

Install extensions from **Marketplace → AreaWP Extensions**. Installed extensions appear as new sections in the admin panel.

## Downloads

**Path:** `/downloads` — Admin only

The Downloads section is the distribution hub for clients to download files you've made available:

- White-label client tools
- PDF guides
- WordPress theme and plugin ZIPs
- License key files

### Making a File Available

1. Go to **Marketplace → Downloads** → **Upload File**
2. Set name, description, and version
3. Choose which pricing plans have access
4. Upload the file
5. Click **Publish**

Clients can download available files from their **Downloads** section.

---

## Managing the Catalog

Admins can curate what appears in the marketplace:

1. Go to **Marketplace → Manage Catalog**
2. Enable or disable marketplace items per category
3. Set visibility by pricing plan
4. Feature specific items (shown at the top)
5. Add custom items (upload your own plugins/themes)

<Tip>
Use the marketplace to centrally distribute your own custom plugins and themes to client sites — avoiding the need to manually upload to each site individually.
</Tip>
