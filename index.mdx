<Info>
**AreaWP** is a modern server control panel that gives agencies, developers, and hosting providers a single interface to manage websites, databases, Docker containers, security, billing, and clients — all from one dashboard.
</Info>

## What is AreaWP?

AreaWP is built for professionals who run VPS or dedicated servers and need a powerful, self-hosted control panel. It replaces a scattered toolset (cPanel, manual CLI, billing software, monitoring dashboards) with a single, cohesive platform.

## Platform Modes

AreaWP supports two deployment modes depending on your business model:

<Tabs>
  <Tab title="SaaS Mode">
    **Sell hosting as a service**

    In SaaS mode, you run a hosted platform where clients sign up, choose a pricing plan, and manage their own sites via the user portal.

    - ✅ Client self-service portal
    - ✅ Subscription billing with Stripe / Bank Transfer
    - ✅ Pricing plans with resource limits
    - ✅ White-label branding
    - ✅ Support center
    - ✅ Service marketplace
  </Tab>
  <Tab title="WaaS Mode">
    **WordPress-as-a-Service**

    In WaaS mode, the platform is optimized for WordPress deployment at scale. Clients subscribe to hosting plans and get managed WordPress sites.

    - ✅ Everything in SaaS mode
    - ✅ WordPress site templates
    - ✅ One-click WordPress deployments
    - ✅ WP pipeline (CI/CD)
    - ✅ Plugin and theme marketplace
    - ✅ Bulk WordPress management
  </Tab>
</Tabs>

## Core Features

<AccordionGroup>
  <Accordion title="Multi-Server Management" icon="server">
    Deploy lightweight agents on any Linux server and manage them all from a single AreaWP dashboard. Monitor CPU, RAM, disk, and network in real time. Provision new cloud servers on DigitalOcean, Hetzner, and other providers directly from the panel.
  </Accordion>

  <Accordion title="Application Hosting" icon="globe">
    Deploy WordPress sites, PHP apps, Node.js applications, and static sites as isolated Docker containers. Each site gets its own container, database, and Nginx config — no shared resources, no cross-site contamination.
  </Accordion>

  <Accordion title="Database Management" icon="database">
    Create and manage MySQL and PostgreSQL databases. Access phpMyAdmin directly from the panel, run queries, manage users, and monitor performance.
  </Accordion>

  <Accordion title="Docker & Containers" icon="box">
    Full Docker container lifecycle management — start, stop, rebuild, view logs, exec into containers. Manage Docker Compose stacks, networks, and volumes from the UI.
  </Accordion>

  <Accordion title="Security & Firewall" icon="shield">
    Built-in UFW firewall management, fail2ban configuration, SSL certificate provisioning (Let's Encrypt), and server hardening tools. Monitor login attempts and blocked IPs in real time.
  </Accordion>

  <Accordion title="Billing & Subscriptions" icon="receipt">
    Full billing system with pricing plans, subscriptions, transactions, and payment gateways (Stripe, Bank Transfer). Automated invoice generation and renewal reminders.
  </Accordion>

  <Accordion title="White-Label Branding" icon="palette">
    Customize the platform with your own brand name, logo, and colors. Present a fully branded experience to your clients without any AreaWP mentions.
  </Accordion>

  <Accordion title="Real-Time Monitoring" icon="chart-line">
    Live metrics via Socket.IO — CPU, memory, disk I/O, network throughput. Set up public status pages to communicate uptime to clients. Configure alerts for resource thresholds.
  </Accordion>
</AccordionGroup>

## Architecture

AreaWP runs as two components:

| Component | Technology | Role |
|-----------|-----------|------|
| **Panel** | Flask (Python) + React | Web dashboard, API, billing, user management |
| **Agent** | Go binary | Installed on each managed server, handles system commands |

The panel communicates with agents over a secure WebSocket connection. Agents execute commands on the server (start/stop services, deploy apps, manage files) and stream real-time metrics back to the panel.

```
┌─────────────┐     HTTPS/WSS     ┌─────────────┐
│  AreaWP     │ ◄────────────────► │   Agent     │
│  Panel      │                   │  (Server 1) │
│  (Flask +   │                   └─────────────┘
│   React)    │     HTTPS/WSS     ┌─────────────┐
│             │ ◄────────────────► │   Agent     │
└─────────────┘                   │  (Server 2) │
                                  └─────────────┘
```
