## Website Packages

Create up to 20 different hosting packages with customizable features and pricing.

### Package Configuration

Each package can include the following settings:

| Setting | Description | Options |
|---------|-------------|---------|
| **Package Name** | Display name for the package | Text (e.g., "Starter", "Pro", "Enterprise") |
| **Price** | Monthly cost | Currency amount |
| **Disk Space** | Storage allocation | 150 - 99,999,999,999 MB |
| **Description** | Package details | Text area for features list |
| **Member Roles** | Who can purchase | Select user roles |
| **Background Colors** | Visual styling | Two color pickers for gradient |
| **Purchase Theme** | Enable theme selection | Toggle on/off |
| **Backup Download** | Allow backup downloads | Toggle on/off |
| **Control Panel Access** | DirectAdmin access | Toggle on/off |
| **Featured Ribbon** | Highlight as recommended | Toggle on/off |

### Creating a Package

<Steps>
  <Step title="Add New Package">
    Click the "Add Package" button to create a new hosting plan
  </Step>
  <Step title="Set Basic Info">
    Enter package name, price, and disk space allocation
  </Step>
  <Step title="Write Description">
    Add a compelling description highlighting key features
  </Step>
  <Step title="Configure Features">
    Toggle features like backup download, control panel access, etc.
  </Step>
  <Step title="Set Visual Style">
    Choose background colors and enable featured ribbon if desired
  </Step>
  <Step title="Select User Roles">
    Choose which member roles can purchase this package
  </Step>
  <Step title="Save Package">
    Save your configuration to make the package available
  </Step>
</Steps>

## Renewal Terms

Configure billing periods and renewal pricing for your packages.

### Available Renewal Periods

Set pricing for different billing cycles:

| Billing Period | Typical Discount | Use Case |
|----------------|------------------|----------|
| **Monthly** | 0% (base price) | Flexibility, testing |
| **Quarterly** | 5-10% | Short-term commitment |

| **Annual** | 15-25% | Best value, customer retention |
| **Biennial** | 20-30% | Long-term commitment |
| **Triennial** | 25-35% | Maximum savings |

## Global Service Pricing

Configure pricing for additional services available across all packages.

### Service Options

**SSL Certificate**

Secure socket layer certificate for HTTPS.

- **Typical Price**: $10 - $30/year
- **Recommended**: Include free with premium packages

**Website Backup**

Automated backup service with restore capability.

- **Typical Price**: $5 - $15/month
- **Recommended**: Include with professional and enterprise packages

**Disk Space Upgrade**

Additional storage beyond package allocation.

- **Typical Price**: $1 - $5 per GB/month
- **Recommended**: Offer in increments (1GB, 5GB, 10GB)

### Pricing Configuration

| Service | Price | Billing | Notes |
|---------|-------|---------|-------|
| **SSL Certificate** | $15 | Annual | Or free with premium plans |
| **Backup Service** | $10 | Monthly | Automated daily backups |
| **Disk Upgrade (1GB)** | $2 | Monthly | Per additional GB |
| **Disk Upgrade (5GB)** | $8 | Monthly | Bulk discount |
| **Disk Upgrade (10GB)** | $15 | Monthly | Best value |

<Tip>
**Service Pricing Tips:**
- Bundle services with higher-tier packages
- Offer volume discounts for bulk purchases
- Make SSL certificates free to encourage security
- Price backups to cover actual storage costs
- Consider offering free trials for services
</Tip>

## Domain Pricing

Set registration and renewal prices for each domain extension (TLD).

### Domain Pricing Structure

For each TLD, you can configure:

| Setting | Description | Example |
|---------|-------------|---------|
| **Registration Price** | Cost to register new domain | $12.99/year |
| **Registration VAT** | Tax on registration | $2.60 (20%) |
| **Renewal Price** | Cost to renew domain | $14.99/year |
| **Renewal VAT** | Tax on renewal | $3.00 (20%) |

### Common TLD Pricing

**Popular TLDs:**

| TLD | Registration | Renewal | Notes |
|-----|--------------|---------|-------|
| .com | $12.99 | $14.99 | Most popular |
| .net | $14.99 | $16.99 | Network sites |
| .org | $13.99 | $15.99 | Organizations |
| .io | $34.99 | $39.99 | Tech startups |
| .co | $24.99 | $29.99 | Companies |

**Premium TLDs:**

| TLD | Registration | Renewal | Notes |
|-----|--------------|---------|-------|
| .app | $18.99 | $22.99 | Applications |
| .dev | $15.99 | $18.99 | Developers |
| .tech | $24.99 | $29.99 | Technology |
| .store | $19.99 | $24.99 | E-commerce |
| .online | $9.99 | $12.99 | Budget option |

### Pricing Strategy

**Cost-Plus Pricing:**
1. Check wholesale cost from registrar (e.g., Dynadot)
2. Add your margin (typically 20-40%)
3. Round to .99 pricing for psychology
4. Set renewal slightly higher than registration

**Example Calculation:**
- Wholesale cost: $8.50
- Your margin: 40% ($3.40)
- Base price: $11.90
- Rounded price: $12.99

<Warning>
**Domain Pricing Considerations:**
- Don't price too low - domains have real costs
- Renewal prices should cover long-term costs
- Consider currency fluctuations
- Monitor competitor pricing
- Update prices annually
</Warning>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Packages Not Displaying" icon="eye-slash">
    **Issue:** Hosting packages don't appear on website
    
    **Common Causes:**
    - Package not saved properly
    - Member role restrictions
    - Cache not cleared
    - Package marked as inactive
    
    **Solutions:**
    1. Verify package is saved and active
    2. Check member role settings
    3. Clear all caches
    4. Review package visibility settings
    5. Check theme compatibility
  </Accordion>
  
  <Accordion title="Incorrect Pricing" icon="dollar-sign">
    **Issue:** Wrong prices showing to customers
    
    **Common Causes:**
    - Pricing not updated
    - Tax calculation errors
    - Currency conversion issues
    - Cached pricing data
    
    **Solutions:**
    1. Update pricing and save
    2. Verify tax settings
    3. Check currency configuration
    4. Clear pricing cache
    5. Test in incognito mode
  </Accordion>
  
  <Accordion title="Domain Pricing Not Working" icon="globe">
    **Issue:** Domain prices not applying correctly
    
    **Common Causes:**
    - TLD not configured
    - VAT settings incorrect
    - Price not saved
    - Domain API issues
    
    **Solutions:**
    1. Configure pricing for all TLDs
    2. Verify VAT percentages
    3. Save domain pricing again
    4. Check domain API connection
    5. Review transaction logs
  </Accordion>
  
  <Accordion title="Addons Not Available" icon="puzzle-piece">
    **Issue:** Custom addons not showing at checkout
    
    **Common Causes:**
    - Addon not enabled
    - Package compatibility
    - Visibility settings
    - Cache issues
    
    **Solutions:**
    1. Enable addon in settings
    2. Check package compatibility
    3. Review visibility rules
    4. Clear cache
    5. Test checkout process
  </Accordion>
</AccordionGroup>
