## Contact Button

### Contact Form Settings

<AccordionGroup>
  <Accordion title="Enable Contact Form Request Callback" icon="envelope">
    Toggle to allow visitors to request callbacks through a contact form. This feature helps capture leads and improve customer engagement.
  </Accordion>
  
  <Accordion title="Enable Helpdesk & Tickets System" icon="headset">
    Enable the helpdesk ticketing system for customer support. When enabled, the helpdesk menu will appear in the admin navigation.
  </Accordion>
</AccordionGroup>

### Contact Buttons Management

Create up to **20 customizable contact buttons** with a sortable, drag-and-drop interface.

<ParamField path="title" type="string" required>
  Button display name (e.g., Facebook, WhatsApp, Telegram)
</ParamField>

<ParamField path="icon" type="image">
  Upload a custom icon for the button
</ParamField>

<ParamField path="link" type="url" required>
  Contact link URL (e.g., `https://wa.me/1234567890`)
</ParamField>

<ParamField path="background" type="color" default="#007bff">
  Custom background color for the button
</ParamField>

<ParamField path="newTab" type="boolean" default={true}>
  Open link in a new browser tab
</ParamField>

### Popular Contact Options

<CodeGroup>
```text WhatsApp
https://wa.me/1234567890
```

```text Facebook Messenger
https://m.me/username
```

```text Telegram
https://t.me/username
```

```text Email

```
</CodeGroup>

<Tip>
**Best Practices:**
- Use recognizable brand colors (WhatsApp: `#25D366`, Facebook: `#1877F2`, Telegram: `#0088CC`)
- Keep button titles short and clear
- Test all links before going live
- Limit to 3-5 contact options for better UX
</Tip>
