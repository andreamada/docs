## Troubleshooting

**License Validation Fails:**
- Verify key is correct (no extra spaces)
- Check expiration date
- Ensure domain matches license
- Contact support if issue persists

**Cron Jobs Not Running:**
- Verify cron job configured on server
- Check cron URL is correct
- Review server logs
- Test URL manually in browser

**Contact Buttons Not Appearing:**
- Save configuration again
- Clear browser and server cache
- Check theme compatibility
- Verify JavaScript has no errors
