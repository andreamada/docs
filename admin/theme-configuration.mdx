## Logo & Branding

Upload custom logos to personalize your admin dashboard.

### Logo Types

**Main Logo**

Primary logo displayed in the admin panel header.

- **Recommended Size**: 180 x 50 pixels
- **Format**: PNG, SVG (transparent background recommended)
- **Location**: Top left of admin dashboard
- **Use Case**: Full color logo for light backgrounds

**Favicon**

Small icon displayed in browser tabs.

- **Recommended Size**: 32 x 32 pixels or 64 x 64 pixels
- **Format**: ICO, PNG
- **Location**: Browser tab
- **Use Case**: Simple, recognizable icon version of your logo

**Login Page Logo**

Logo displayed on the admin login screen.

- **Recommended Size**: 200 x 80 pixels
- **Format**: PNG, SVG
- **Location**: Center of login page
- **Use Case**: Can be larger, more detailed version

**Sidebar Logo**

Compact logo for collapsed sidebar.

- **Recommended Size**: 40 x 40 pixels (square)
- **Format**: PNG, SVG
- **Location**: Sidebar when collapsed
- **Use Case**: Icon or simplified logo mark

## Color Scheme

Customize the admin panel colors to match your brand.

### Color Settings

**Primary Color**

Main brand color used throughout the admin panel.

- **Used For**: Buttons, links, active states, highlights
- **Format**: Hex color code (e.g., `#3B82F6`)
- **Recommendation**: Use your main brand color

**Secondary Color**

Accent color for secondary elements.

- **Used For**: Secondary buttons, badges, accents
- **Format**: Hex color code (e.g., `#8B5CF6`)
- **Recommendation**: Complementary to primary color

**Sidebar Background**

Background color for the navigation sidebar.

- **Used For**: Sidebar background
- **Format**: Hex color code (e.g., `#1F2937`)
- **Common Options**: Dark gray, navy, black, or brand color

**Sidebar Text Color**

Text color for sidebar menu items.

- **Used For**: Menu item text
- **Format**: Hex color code (e.g., `#FFFFFF`)
- **Recommendation**: High contrast with sidebar background

**Header Background**

Background color for the top header bar.

- **Used For**: Top navigation bar
- **Format**: Hex color code (e.g., `#FFFFFF`)
- **Common Options**: White, light gray, or brand color

### Color Scheme Examples

**Professional Blue:**
- Primary: `#3B82F6` (Blue)
- Secondary: `#10B981` (Green)
- Sidebar BG: `#1E293B` (Dark slate)
- Sidebar Text: `#F1F5F9` (Light gray)
- Header BG: `#FFFFFF` (White)

**Modern Dark:**
- Primary: `#8B5CF6` (Purple)
- Secondary: `#EC4899` (Pink)
- Sidebar BG: `#111827` (Very dark gray)
- Sidebar Text: `#E5E7EB` (Light gray)
- Header BG: `#1F2937` (Dark gray)

**Clean Minimal:**
- Primary: `#0EA5E9` (Sky blue)
- Secondary: `#F59E0B` (Amber)
- Sidebar BG: `#F9FAFB` (Very light gray)
- Sidebar Text: `#374151` (Dark gray)
- Header BG: `#FFFFFF` (White)

<Warning>
**Color Accessibility:**
- Ensure sufficient contrast between text and backgrounds
- Test colors for readability
- Consider colorblind users
- Follow WCAG accessibility guidelines
- Avoid using color alone to convey information
</Warning>

## Sidebar Configuration

Customize the navigation sidebar appearance and behavior.

### Sidebar Settings

**Sidebar Style**

Choose how the sidebar appears.

| Style | Description | Use Case |
|-------|-------------|----------|
| **Fixed** | Sidebar always visible | Desktop users, large screens |
| **Collapsible** | Can be expanded/collapsed | Flexible, more screen space |
| **Mini** | Always shows icons only | Maximum content space |
| **Overlay** | Slides over content | Mobile-friendly |

**Default State**

Set whether sidebar starts expanded or collapsed.

- **Expanded**: Full sidebar with text and icons (default)
- **Collapsed**: Icons only, expands on hover
- **Recommendation**: Expanded for desktop, collapsed for tablets

**Sidebar Position**

Choose sidebar location.

- **Left**: Standard position (recommended)
- **Right**: Alternative layout

### Menu Organization

**Menu Items**

Customize which menu items appear in the sidebar:

- Dashboard
- Users
- Websites
- Domains
- Billing
- Support Tickets
- Reports
- Settings

**Menu Grouping**

Organize menu items into logical groups:

- **Management**: Users, Websites, Domains
- **Financial**: Billing, Invoices, Reports
- **Support**: Tickets, Documentation
- **System**: Settings, Logs, API

<Tip>
**Sidebar Best Practices:**
- Keep menu items organized and logical
- Use icons for better visual recognition
- Limit top-level items to 8-10 for clarity
- Group related items together
- Show most-used items first
</Tip>

## Additional Settings

### Date & Time Format

**Date Format**

Choose how dates are displayed throughout the admin panel.

| Format | Example | Region |
|--------|---------|--------|
| `MM/DD/YYYY` | 12/31/2024 | US |
| `DD/MM/YYYY` | 31/12/2024 | UK, Europe |
| `YYYY-MM-DD` | 2024-12-31 | ISO Standard |
| `DD MMM YYYY` | 31 Dec 2024 | Readable |

**Time Format**

Choose 12-hour or 24-hour time display.

- **12-hour**: 2:30 PM
- **24-hour**: 14:30

**Timezone**

Set the default timezone for the admin panel.

- **Recommendation**: Use your business location timezone
- **Note**: Affects all timestamps and scheduled tasks

### Language Settings

**Admin Panel Language**

Choose the language for the admin interface.

- Default: English
- Available: Multiple languages supported
- Note: Separate from customer-facing language settings

### Session Settings

**Session Timeout**

How long admins can be inactive before automatic logout.

- **Range**: 15 minutes - 24 hours
- **Recommended**: 30-60 minutes for security
- **Note**: Shorter timeout = better security

**Remember Me**

Allow admins to stay logged in across sessions.

- **Enabled**: Checkbox on login page
- **Disabled**: Must log in each time
- **Recommendation**: Enable for trusted devices only

## Troubleshooting

<AccordionGroup>
  <Accordion title="Logo Not Displaying" icon="image">
    **Issue:** Uploaded logo doesn't appear
    
    **Common Causes:**
    - File size too large
    - Unsupported file format
    - Incorrect dimensions
    - Cache not cleared
    
    **Solutions:**
    1. Compress image file (< 500KB)
    2. Use PNG or SVG format
    3. Resize to recommended dimensions
    4. Clear browser cache
    5. Hard refresh (Ctrl+F5)
    6. Check file permissions
  </Accordion>
  
  <Accordion title="Colors Not Applying" icon="palette">
    **Issue:** Custom colors not showing
    
    **Common Causes:**
    - Invalid hex color code
    - Cache not cleared
    - Settings not saved
    - CSS conflicts
    
    **Solutions:**
    1. Verify hex codes are valid (e.g., #3B82F6)
    2. Save settings again
    3. Clear application cache
    4. Clear browser cache
    5. Test in incognito mode
    6. Check for CSS overrides
  </Accordion>
  
  <Accordion title="Sidebar Not Collapsing" icon="bars">
    **Issue:** Sidebar won't collapse or expand
    
    **Common Causes:**
    - JavaScript error
    - Browser compatibility
    - Settings conflict
    - Cache issue
    
    **Solutions:**
    1. Check browser console for errors
    2. Try different browser
    3. Clear browser cache
    4. Disable browser extensions
    5. Check sidebar settings
    6. Reset to default settings
  </Accordion>
  
  <Accordion title="Dashboard Widgets Missing" icon="table-columns">
    **Issue:** Widgets not appearing on dashboard
    
    **Common Causes:**
    - Widgets disabled in settings
    - Permission issues
    - Data loading errors
    - Cache problems
    
    **Solutions:**
    1. Check widget visibility settings
    2. Verify user permissions
    3. Review error logs
    4. Clear dashboard cache
    5. Reset widget layout
    6. Check data sources
  </Accordion>
</AccordionGroup>
